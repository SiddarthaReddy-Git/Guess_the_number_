# Guess the Number 

This is a simple Python project where the computer randomly generates a number, and the player has five chances to guess it correctly.
The program gives feedback after each attempt, telling the user whether their guess is too high or too low, until they either succeed or run out of attempts.  

The project demonstrates the use of Python basics such as loops, conditionals, and the `random` module.
It is a beginner‑level game designed to strengthen understanding of user input handling and control flow in Python.  

Future improvements could include adding difficulty levels, tracking scores across multiple rounds, or even building a graphical interface using Tkinter or PyQT.
This project is a good starting point for anyone learning Python and wanting to practice problem‑solving with simple logic.
